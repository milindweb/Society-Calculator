v1.1.0 · Production Release

About the Application

Society Calculator is a lightweight offline desktop application designed to simplify the day-to-day financial operations of Co-operative Credit Societies. It performs society-specific financial calculations quickly and accurately while maintaining a simple and user-friendly interface. The application works completely offline and stores all records locally using a SQLite database. No internet connection or external server is required.
