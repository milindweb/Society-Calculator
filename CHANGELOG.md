# Changelog

## v1.0.0 (2026-06-28)

### Added
- Initial release of Society Calculator
- Loan Sanction Calculator with amortization schedule
- Final Settlement Calculator
- Fixed Deposit Calculator with rate slabs
- Simple Calculator for quick office calculations
- Saved Records with search, view, edit, and delete
- Configurable Settings (interest rate, loan limits, FD rates, etc.)
- SQLite local database storage
- PyWebView desktop application

### Changed
- Navigation tabs renamed to short labels: LOAN, FS, FD, CAL, DATA, SETTING
- Added "SOCIETY CALCULATOR" header above navigation
- About section moved from tab to page footer
- Mobile responsive layout with hamburger menu toggle
