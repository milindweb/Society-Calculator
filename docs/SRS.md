# 1. Introduction

## 1.1 Purpose

Society Calculator is a lightweight offline desktop application developed to simplify the day-to-day financial operations of Co-operative Credit Societies.

The application is designed specifically for society office staff who currently perform calculations manually using Microsoft Excel.

The objective of this software is to reduce manual calculations, improve accuracy, save every calculation automatically, and provide a simple desktop interface that works completely offline.

The application is developed using **Python** and **SQLite**, ensuring fast performance, reliable local data storage, and complete offline functionality without requiring an internet connection.

All records are stored securely on the local computer and can be searched, viewed, edited, and managed easily.

The software is intentionally kept simple. No unnecessary modules or enterprise features are included.

The application shall support a responsive desktop interface that adapts to different window sizes. Users should be able to use the application in a compact window similar to the Windows Calculator for quick calculations or maximize it to full-screen mode for a larger workspace. All forms, tables, buttons, and input fields shall automatically resize and reposition to utilize the available space while maintaining a clean, organized, and user-friendly layout. The application shall remain fully functional and readable in both compact and maximized views without requiring separate layouts or reducing usability.

The application contains the following modules:

# 3.1 Final Settlement Module (Section 4.1)

## Input Fields

| Field               | Type         |
|---------------------|--------------|
| Name                | Text         |
| Generation No       | Text         |
| Short Loan          | Decimal (₹)  |
| Long Loan           | Decimal (₹)  |
| Long Loan Interest  | Decimal (₹)  |
| Short Loan Interest | Decimal (₹)  |
| Other Deduction     | Decimal (₹)  |
| CD (Earning)        | Decimal (₹)  |
| Share (Earning)     | Decimal (₹)  |
| DCRB (Earning)      | Decimal (₹)  |
| Other Earning       | Decimal (₹)  |
| Remark              | Text         |

## Calculations

- **Total Deduction** = Short Loan + Long Loan + Long Loan Interest + Short Loan Interest + Other Deduction
- **Total Earning** = CD + Share + DCRB + Other Earning
- **Final Settlement Amount** = Total Deduction − Total Earning
- **Status**:
  - If Final Amount > 0 → **Member Pays**
  - If Final Amount < 0 → **Society Pays**

## Actions

- **[Calculate]** — Performs above calculations and shows result.
- **[Save]** — Persists the settlement record.
- **[Clear]** — Resets all input fields.

## Formulas

### CD Deduction

```
CD Deduction = Required CD - Present CD
```

### Share Deduction

```
Share Deduction = Required Share - Present Share
```

### Total Deduction

```
Total Deduction =
CD Deduction
+ Share Deduction
+ Old Long Loan
+ Old Short Loan
+ Other Deduction
```

---

## Loan Sanction

### New Loan Amount

User Input

### Amount In Hand

```
Amount In Hand =
New Loan Amount - Total Deduction
```

---

## Loan Settings

### Interest Rate

Default = **11.25%** (editable)

### Principal Recovery Per EMI

User Input (e.g. ₹5,000)

---

## EMI Calculation

### Monthly Interest

```
Monthly Interest =
Outstanding Balance × Interest Rate ÷ 12
```

### Monthly EMI

```
Monthly EMI =
Principal Recovery + Monthly Interest
```

### New Outstanding Balance

```
Outstanding Balance =
Previous Balance - Principal Recovery
```

Repeat until balance becomes ₹0.

---

## Total Interest

```
Total Interest =
Sum of all Monthly Interest
```

---

## Total Repayment

```
Total Repayment =
New Loan Amount + Total Interest
```

# 3.3 Fixed Deposit Calculator Module

## Input Fields

| Field             | Type         |
|-------------------|--------------|
| FD No.            | Text         |
| Name              | Text         |
| FD Amount         | Decimal (₹)  |
| Duration (Months) | Integer      |
| Interest Rate     | Decimal (%)  |

## Rate Slabs (Default)

| Tenure              | Rate |
|---------------------|------|
| 1–360 Days          | 4.0% |
| 12 Months (1 Year)  | 8.0% |
| 24 Months (2 Year)  | 8.5% |
| 36 Months (3 Year)  | 9.5% |

## Calculations

- **Total Interest** = FD Amount × Rate ÷ 100 × (Months ÷ 12)
- **Total Amount** = FD Amount + Total Interest

## Display

| FD No. | Name | FD Amount | Months | Interest % | Total Interest | Total Amount |

Rate slab auto-suggest based on tenure selection.

## Actions

- **[Calculate]** — Computes interest and maturity amount.
- **[Save]** — Persists FD record.
- **[Clear]** — Resets all fields.
