About Society Calculator

Society Calculator is a lightweight offline desktop application developed to simplify the day-to-day financial operations of Co-operative Credit Societies.

The software is designed to perform society-specific financial calculations quickly and accurately while maintaining a simple and user-friendly interface.

The application works completely offline and stores all records locally using a SQLite database. No internet connection or external server is required.

Technology Used

  • Python 3
  • SQLite Database
  • HTML / CSS / JavaScript
  • PyWebView (Desktop)
  • PyInstaller (Windows & Linux)

Key Features

  • Offline Desktop Application
  • Loan Sanction Calculator
  • Final Settlement Calculator
  • Fixed Deposit Calculator
  • Simple Calculator
  • Automatic Amortization Schedule
  • Search Saved Records
  • Edit & Delete Saved Records
  • Fast SQLite Database
  • Lightweight and Easy to Use
